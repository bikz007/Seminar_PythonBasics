Understanding string with python

Almost every useful program involves some kind of text processing, whether it is parsing data or generating output.
When programming, text is represented by strings.

What is String in Python?

A string is usually a bit of text you want to display to someone, or "export" out of the program you are writing. Python knows you want something to be a string when you put either " (double-quotes) or ' (single-quotes) around the text.

Example 1

               				
	       In  [1]:str1="Hello world.This is a string"

	       In  [2]: str2='Hello python.This is string2 with single quote'

								In  [3]: str1

								Out [3]: 'Hello world.This is a string'

								In  [4]: str2

								Out [4]: 'Hello python.This is a string 2 with single quote'

								In  [5]: #In python we use 'print' function to print the string in console
									print(str1)

								Out [5]:  Hello world.This is a string

String values can be enclosed in either single or double quotes. The two different kinds of quotes work the same way, but having both allows you to include one kind of quotes inside of a string specified with the other kind of quotes, without needing to escape them with the backslash character:

'isn\'t that grand'
"isn't that grand"

The main tool Python gives us to process text is stringsâ€”immutable sequences of characters.

There are actually two kinds of strings:

  • Plain strings, which contain 8-bit (ASCII) characters
  • Unicode strings, which contain Unicode characters
		               				
								       In  [1]:str1="Hello world.This is a string"

								       In  [2]: str2='Hello python.This is string2 with single quote'
str1 and str2 is an example of plain string

	               				
	       In   [1]: #Unicode string
			   			 hello = u'Hello\u0020World'

								In   [2]: hello

								Out  [2]: 'Hello World'
hello is an example of unicode string
We can precede a string literal with a u or U to make it a Unicode string.

To have a string literal span multiple lines, we can use a backslash as the last character on the line, which indicates that the next line is a continuation.

		               				
									In  [1]:bigstring1 = "This is a long string\
									        that spans two lines."

									In  [2]:print(bigstring1)

									Out [2]:This is a long string that spans two lines.

We must embed newlines in the string if you want the string to output on two lines:

		               				
									In  [1]: bigstring2 = "This is a long string\n\
										that prints on two lines."

									In  [2]: print(bigstring2)

									Out [2]: This is a long string
										that prints on two lines.

Another approach is to enclose the string in a pair of matching triple quotes (either single or double):

		               				
		       In  [1]: biggerstring = """
										This is an even
										bigger string that
										spans three lines.
										"""

									In  [2]: print(biggerstring)

									Out [2]:This is an even
										bigger string that
										spans three lines.

Some String Functions

		               				
		       In  [1]:string1="Python programming is super easy"

									In  [2]:string1.upper()

									Out [2]:'PYTHON PROGRAMMING IS SUPER EASY'

									In  [3]:string1.lower()

									Out [3]:'python programming is super easy'

									In  [4]:#strip function
										#Return a copy of the string S with leading and trailing
										#    whitespace removed.
										#   If chars is given and not None, remove characters in chars instead.

										"AABAA".strip("A")

									Out [4]:'B'

									In  [5]:"ABCABBA".strip("AB")

									Out [5]:'C'

									In  [6]:# S.split(sep=None, maxsplit=-1) -> list of strings
										    
										#     Return a list of the words in S, using sep as the
										#     delimiter string.  If maxsplit is given, at most maxsplit
										#     splits are done. If sep is not specified or is None, any
										#     whitespace string is a separator and empty strings are
										#     removed from the result.

										x = "blue,red,green"
										x.split(',')

									Out [6]:['blue', 'red', 'green']

									In  [7]:# S.partition(sep) -> (head, sep, tail)
										    
										#     Search for the separator sep in S, and return the part before it,
										#     the separator itself, and the part after it.  If the separator is not
										#     found, return S and two empty strings.

										biggerstring = """
										This is an even
										bigger string that
										spans three lines.
										"""
										biggerstring.partition('bigger string')

									Out [7]:('\nThis is an even\n', 'bigger string', ' that\nspans three lines.\n')
